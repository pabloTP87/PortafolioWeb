# Portafolio web
Portafolio personal para mostrar los distintos proyectos realizados a lo
largo del tiempo, principalmente en el ambito del desarrollo móvil, como también, 
desarrollos con tecnologías web.

## Tecnologías utilizadas:

* HTML 5
* CSS 3
* Gulp.js
* Sass
* Adobe XD para prototipado
* Adobe Illustrator para ilustraciones y diseños vectoriales.
